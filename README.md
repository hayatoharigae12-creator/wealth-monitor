・WealthMonitor

Gmail経由で銀行の利用通知を自動解析し、視認性の高いダッシュボードで支出を可視化するフルスタックWebアプリです。

主な機能
- **自動データマイニング**: Gmail APIを使用して、銀行からの利用通知メールから金額・日付を自動抽出。
- **ハイパフォーマンス同期**: 差分更新ロジックにより、新着メールのみを解析。リロード速度を大幅に高速化。
- **マルチビュー・ダッシュボード**: 曜日別・時間帯別の支出を「金額」と「回数」の2軸で分析可能。
- **モダンUI**: ダークモードに最適化された、直感的でレスポンシブなインターフェース。

使用技術
- Frontend/Backend: Next.js (App Router), TypeScript
- Auth/API: Google OAuth2, Gmail API
- Styling: CSS-in-JS (Inline Styles)

独自のこだわり
大量のメールを解析する際、APIの通信速度がボトルネックとなりました。これを解決するため、ローカルにJSONデータベースを構築し、保存済みデータのIDと照合して新着分のみをフェッチする「スマート同期ロジック」を自作しました。その結果、リロード時間を約90%短縮することに成功しました。
